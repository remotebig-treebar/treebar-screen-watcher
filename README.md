# Treebar Screen Watcher — คู่มือติดตั้ง

## วิธีที่ง่ายที่สุด: ใช้ GitHub Actions build .exe ให้อัตโนมัติ (ฟรี)

### ขั้นตอน (ทำครั้งเดียว 5 นาที)

1. สร้าง GitHub account ฟรีที่ https://github.com
2. สร้าง repository ใหม่ชื่อ treebar-screen-watcher (Public)
3. Upload ไฟล์ทั้งหมดที่ส่งมาเข้า repository
4. ไปที่ tab Actions รอ build เสร็จ (3-5 นาที)
5. ดาวน์โหลด .exe จาก tab Releases แล้วดับเบิลคลิกติดตั้ง

## วิธีใช้งาน

1. เปิด Treebar Screen Watcher จาก Desktop
2. กรอก Claude API Key (จาก console.anthropic.com)
3. กด "เลือกบริเวณบนหน้าจอ" ลากคลุม LINE/Krungthai Connext
4. กด "เริ่มจับภาพ"
